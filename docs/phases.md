## Project Phases

### Phase 0 – Documentation & Ethics
Define scope, risks, and learning objectives.

### Phase 1 – Insecure Authentication
Implement plaintext password storage to demonstrate weaknesses.

### Phase 2 – Secure Authentication
Apply hashing and salting to protect credentials.

### Phase 3 – Attack Simulation
Simulate basic offline attacks and compare security impact.

### Phase 4 – Reflection
Analyze improvements, limitations, and ethical considerations.
